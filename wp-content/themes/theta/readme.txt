=== Description ===
Theta is a fully responsive one page wordpress theme which has a modern design with clean lines and style. Excellent prebuilt sections can easily and elegantly display all your important content on a single page. Theta will make your online business a professional and eye-catching look.

=== Installation ===
1. Primary: Login to your wp-admin area and go to Appearance -> Themes. Select Install tab and click on Upload link. Select theme .zip and click on Install now button. If you have any errors, use alternate method.
2. Alternate: Unzip the template file (Theta.zip) that you have downloaded. Via FTP, upload the whole folder (Theta) to your server and place it in the /wp-content/themes/ folder. Do not change directory name. The template files should thus be here now: /wp-content/themes/Theta/index.php (for example).
3. Log into your WP admin panel and click on the Design tab. Now click on the Theta theme to activate it.
